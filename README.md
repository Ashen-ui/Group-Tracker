# Group Tracker

Projet de serveur web en Go, dans lequel nous devons utiliser une API.

Pour démarrer le projet, il faudra écrire cette commande dans le terminal.

```bash
go run main.go
```
Le serveur sera accessible sur `http://localhost:8081`

Bienvenue sur Groupie Tracker, une application web permettant de rechercher des artistes et leurs concerts.
